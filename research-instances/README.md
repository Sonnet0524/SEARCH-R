# Research Instances - 研究实例注册表

本目录用于注册使用SEARCH-R方法论创建的研究课题。

## 📋 用途

- 记录使用SEARCH-R方法论的研究课题
- 提供研究实例的索引和引用
- 跟踪研究课题的进展

## 📁 结构说明

```
research-instances/
├── README.md              # 本文件：管理说明
└── {project-name}.md      # 研究课题引用文件
```

## 📝 注册格式

每个研究课题创建一个简单的引用文件：

```markdown
# 项目名称

**仓库地址**: [项目GitHub地址]
**研究主题**: 简要描述
**状态**: Active/Paused/Completed
**创建时间**: YYYY-MM-DD
```

## 🔍 使用说明

### 注册新课题

当你使用SEARCH-R创建新的研究课题时：

1. 创建独立的研究项目仓库
2. 在本目录创建引用文件 `{project-name}.md`
3. 填写项目基本信息

### 更新课题状态

定期更新研究课题的状态：
- Active：正在进行中
- Paused：暂时中断
- Completed：已完成
