# References - 参考资料

本目录存放研究过程中收集的参考资料。

## 📁 目录结构

```
references/
├── academic-papers/      # 学术论文
├── official-documents/   # 官方文件
├── industry-reports/     # 行业报告
├── news-articles/        # 新闻报道
└── other-materials/      # 其他资料
```

## 📋 使用说明

### 1. 收集资料

根据研究课题需要，收集相关参考资料：

**学术论文**：
- 知网、万方、Web of Science等数据库
- Google Scholar、arXiv等平台
- 相关领域的顶级会议和期刊

**官方文件**：
- 政府政策文件
- 行业标准规范
- 企业官方报告

**行业报告**：
- 咨询公司报告
- 行业分析报告
- 市场调研报告

**新闻报道**：
- 主流媒体报道
- 行业媒体文章
- 深度调查报道

### 2. 整理资料

**命名规范**：
```
[年份]-[来源]-[标题简称].pdf
[年份]-[作者]-[标题简称].pdf
```

示例：
```
2024-Nature-LLM-Reasoning.pdf
2024-OpenAI-GPT4-Technical-Report.pdf
```

**元数据记录**：
为每份重要资料创建一个简短的README，记录：
- 来源和出处
- 获取时间
- 核心观点摘要
- 与研究课题的关系

### 3. 引用资料

在研究文档中引用资料时：

```markdown
根据 [Nature 2024] 的研究，LLM的推理能力...

参见参考资料：[OpenAI GPT-4技术报告](references/official-documents/2024-OpenAI-GPT4-Technical-Report.pdf)
```

## 📚 资料分类示例

### academic-papers/
学术论文，按主题或年份组织

### official-documents/
- 政策文件
- 标准规范
- 企业报告

### industry-reports/
- 市场分析
- 技术趋势
- 行业白皮书

### news-articles/
重要新闻报道和深度文章

### other-materials/
其他无法归类的资料

## 🔍 资料检索

### 全文搜索
使用工具进行全文搜索：
- macOS: Spotlight
- Linux: grep, ripgrep
- Windows: Everything

### 文献管理
推荐使用文献管理工具：
- Zotero（免费开源）
- Mendeley
- EndNote

## 📝 资料评估

收集资料时，评估其质量：

**权威性**：
- 来源是否可靠？
- 作者/机构是否权威？

**时效性**：
- 发表时间是否合适？
- 内容是否过时？

**相关性**：
- 与研究课题的相关程度？
- 是否提供关键信息？

**质量**：
- 研究方法是否严谨？
- 论证是否充分？
