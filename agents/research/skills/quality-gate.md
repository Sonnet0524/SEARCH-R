# 质量门控能力

> 评估研究质量和判断Human介入时机的能力

---

## 📋 能力定义

评估研究成果的质量，判断研究的确定性和可接受性，决定何时需要Human介入。

## 🎯 使用场景

- 需要判断研究结论是否确定
- 需要评估研究成果是否可接受
- 需要决定是否呼叫Human
- 需要进行质量门控判断

## 🔄 工作流程

### Step 1: 确定性评估

```
HIGH: 逻辑清晰，证据充分
MEDIUM: 逻辑基本清晰，证据部分缺失
LOW: 逻辑不清晰或证据严重缺失
```

### Step 2: 可接受性评估

```
HIGH: 符合预期，可直接应用
MEDIUM: 基本符合预期，需要小幅调整
LOW: 不符合预期，需要重大调整
```

### Step 3: 混淆判断

```
无混淆: 理解一致，无偏差
有混淆: 存在理解偏差或歧义
```

### Step 4: 综合判断

```
IF 确定性=HIGH AND 可接受性=HIGH AND 无混淆
THEN Agent自主决策
ELSE 呼叫Human
```

## 📊 评估标准

- **准确性**: 评估是否准确反映实际情况？
- **及时性**: 是否在关键时刻进行评估？
- **有效性**: 决策是否有效推进研究？
- **效率性**: 是否实现了Human参与最小化？

## 📖 评估模板

```markdown
# 质量门控评估

## 评估对象
[对象描述]

## 确定性评估
- 等级: [HIGH/MEDIUM/LOW]
- 理由: [理由说明]

## 可接受性评估
- 等级: [HIGH/MEDIUM/LOW]
- 理由: [理由说明]

## 混淆判断
- 结果: [有/无]
- 说明: [说明]

## 综合判断
- 决策: [自主决策/呼叫Human]
- 行动: [具体行动]
```
